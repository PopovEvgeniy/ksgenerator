                   The kms activation script generator for Microsoft Windows by Popov Evgeniy Alekseyevich

System requirements

Windows Vista or higher

Program description

This software lets you create a activation script. It will activate Microsoft Windows via a custom KMS server.
Please use this script for non-commercial home purposes only.

Important remark

The activation script needs admin rights for correct work. Run it as an admin.

Product key

You require a volume license key.
You can take the keys for the modern version of Microsoft Windows here: https://docs.microsoft.com/en-us/windows-server/get-started/kmsclientkeys
You can take the keys for the old version of Microsoft Windows here: https://docs.microsoft.com/en-us/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/jj612867(v%3Dws.11)
